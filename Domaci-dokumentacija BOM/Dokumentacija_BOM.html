1. Event atributi

Event-i ili dogadjaji su pojave koje su rezultat necega sto korisnik uradi.Mogu se pozivati inline (kao HTML atribut se dobija tako što se uz naziv dogadjaja dodaje prefiks “on” tj. on{eventtype}), preko addEventListener-a.

Inline koriscenje za "onclick" event

Pritisni na ovaj tekst i ...

Ovo smo dobili tako sto smo na div dodali inline event (onclick):

<div class="prezentacija" onclick="alert('pojavice se alert');">Pritisni na ovaj tekst i ...</div>

preko "addEventListener"-a za "onclick" event

Pritisni na ovaj tekst i promeni cu boju fonta i pozadinu

Ovo smo dobili tako sto smo na div dodali id=onclick i posle ga pozvali preko funkcije:

<div id="onclick">Pritisni na ovaj tekst i promeni cu boju fonta</div>

<script>
  var onclickDiv = document.getElementById("onclick");
  onclickDiv.addEventListener('click', function () {
      this.style.color = 'yellow';
      this.style.backgroundColor = 'blue';
  })
</script>

Ostale primere cemo raditi preko addEventListener-a jer je najprihvaćeniji način registrovanja dogadjaja i omogućava registrovanje više dogadjaja na jednom elementu.

Ovo smo dobili tako sto smo na input tag dodali id=onblur i posle ga pozvali preko funkcije:

Navigator objekat sadrži informacije o korisnikovom browser-u. Objekat Navigator je nazvan po veb čitaču Netscape Navigator, ali ga podržavaju i svi ostali čitači. Objekat window.navigator se može pisati bez prefiksa window.

Svojstvo navigator.userAgent vraca string koji citac upisuje u zaglavlje USER-AGENT HTTP. Ovo svojstvo obicno sadrzi sve informacije iz svojstava appName i appVersion, a cesto i dodatne detalje. Svojstvo se pise na sledeci nacin:

Svojstvo navigator.javaEnabled ( ) prikazuje da li je Java ukljucena u pregledacu. Svojstvo se pise na sledeci nacin:

Date.now( );

new Date()

Date objekat se pravi preko new Date() konstruktora i on predstavlja lokalno vreme koje je bilo na računaru u trenutku kreiranja objekta. Metoda se pise na sledeci nacin:
new Date( );

Prikazuje četvorocifrenu godinu. Metoda se pise na sledeci nacin:
var d = new Date ();
d.getFullYear();

.getMonth()

Prikazuje redni broj meseca (0-11). Metoda se pise na sledeci nacin:
var d = new Date ();
d.getMonth();

.getDate( )

Prikazuje dan u mesecu (1-31). Metoda se pise na sledeci nacin:
var d = new Date ( );
d.getDate( );

.getDay( )

Prikazuje broj dana u nedelji (0-Nedelja,1-Ponedeljak,2-Utorak,3-Sreda,4-Cetvrtak,5-Petak,6-Subota). Metoda se pise na sledeci nacin:
var d = new Date ( );
d.getDay( );

.getHours( )

Prikazuje sat u formatu (0-23). Metoda se pise na sledeci nacin:
/code> var d = new Date ( );

.getMinutes( )

Prikazuje minute u formatu (0-59). Metoda se pise na sledeci nacin:

d.getMinutes( );

.getSeconds( )

Prikazuje sekunde u formatu (0-59). Metoda se pise na sledeci nacin:
var d = new Date ( );
d.getSeconds( );

Prikazuje milisekunde u formatu (0-999). Metoda se pise na sledeci nacin:
