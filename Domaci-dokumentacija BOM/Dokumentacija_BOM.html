Event atributi

Event-i ili dogadjaji su pojave koje su rezultat necega sto korisnik uradi.Mogu se pozivati inline (kao HTML atribut se dobija tako što se uz naziv dogadjaja dodaje prefiks “on” tj. on{eventtype}), preko addEventListener-a.

Inline koriscenje za "onclick" event

Pritisni na ovaj tekst i ...

Ovo smo dobili tako sto smo na div dodali inline event (onclick):

<div class="prezentacija" onclick="alert('pojavice se alert');">Pritisni na ovaj tekst i ...</div>

preko "addEventListener"-a za "onclick" event

Pritisni na ovaj tekst i promeni cu boju fonta i pozadinu

Ovo smo dobili tako sto smo na div dodali id=onclick i posle ga pozvali preko funkcije:

<div id="onclick">Pritisni na ovaj tekst i promeni cu boju fonta</div>

<script>
  var onclickDiv = document.getElementById("onclick");
  onclickDiv.addEventListener('click', function () {
      this.style.color = 'yellow';
      this.style.backgroundColor = 'blue';
  })
</script>

Ostale primere cemo raditi preko addEventListener-a. jer je najprihvaćeniji način registrovanja dogadjaja i omogućava registrovanje više dogadjaja na jednom elementu.

"onblur" event

Kad napisemo tekst i izadjemo iz polja promenice se bojafonta,pozadina i sva slova ce biti velika

<input id="onblur">Kad napisemo tekst i izadjemo iz polja promenic se boja fonta, pozadina i sva slova ce biti velika.
<script>
  var onblur = document.getElementById("onblur");
  onblur.addEventListener('blur', function () {
      this.style.color = 'yellow';
      this.style.backgroundColor = 'blue';
      onblur.value = onblur.value.toUpperCase();
  })
</script>

"onfocus" event

Kad pisemo tekst promenice se boja fonta i pozadina.

<input id="onfocus">Kad pisemo tekst promenice se boja fonta i pozadina.
<script>
  var onfocus = document.getElementById("onfocus");
  onfocus.addEventListener('focus', function () {
      this.style.color = 'white';
      this.style.backgroundColor = 'red';
  })
</script>
