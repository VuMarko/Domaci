UVOD

Dva glavna svojstva omogućavaju animaciju iz CSS-a: transition i animation.

Transition (Tranzicija) se koristi da ublaži naglu promenu vrednosti izabranog CSS svojstva u određenom vremenskom periodu, tako što postepeno menja vrednosti CSS svojstva. Animation (Animacija) se takodje zasniva na promeni svojstva nekog elementa u toku vremena ali dozvoljava malo bolju kontrolu i ima dodatne druge specifičnosti.

U svojstvu “transition” se animacija izvršava samo na osnovu početnih i krajnjih vrednosti nekog CSS svojstva, dok kod svojstva “animation” imamo mogućnost da kroz @keyframe definišemo neograničen broj medjuvrednosti.

Svojstvo “animation” za razliku od “transition” može da odloži pokretanje animacije.

TRANSFORMACIJE

“Transform” nam omogućava da HTML element:

  • transliramo duž osa
    • transform: translate(12px, 50%)
    • transform: translateX(2em)
    • transform: translateY(3in)
    • transform: translate3d(x, y, z)
  • rotiramo
    • transform: rotateX(10deg)
    • transform: rotateY(10deg)
    • transform: rotateZ(10deg)
    • transform: rotate3d(x, y, z)
  • skaliramo po veličini
    • transform: scale(2, 0.5)
    • transform: scaleX(2)
    • transform: scaleY(0.5)
    • transform: scale3d(sx, sy, sz)
  • iskosimo
    • transform: skew(30deg, 20deg)
    • transform: skewX(30deg)
    • transform: skewY(1.07rad)
  • dodamo perspektivu
    • transform: perspective(17px)

Primeri 2D

Translate()

pomera element u odnosu na trenutnu poziciju.

Rotate()

rotira element u zavisnosti od zadatog ugla.

Scale()

uvećava ili smanjuje veličinu elementa u zavisnosti od parametara

SkewX()

iskosiće element po X osi.

SkewY()

iskosiće element po Y osi.

Matrix()

metoda je kombinacija svih 2D metoda u jednoj i parametri izgledaju ovako: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

Primeri 3D

RotateX()

rotira element po X-osi u zavisnosti od zadatog ugla

RotateY()

rotira element po Y-osi u zavisnosti od zadatog ugla.

ScaleZ()

rotira element po Z-osi u zavisnosti od zadatog ugla.

Spisak svih funkcija izgleda ovako: matrix(n,n,n,n,n,n), translate(x,y), translateX(n), translateY(n), scale(x,y), scaleX(n), scaleY(n), rotate(angle), skew(x-angle,y-angle), skewX(angle), skewY(angle).

3D transform osobine:

  • transform - primenjuje 2D ili 3D transformaciju na element
  • transform-origin - omogućava da promenite poziciju transformisanog elementa
  • transform-style - određuje kako da element bude prikazan u 3D prostoru
  • perspective - određuje perspektivu
  • perspective-origin - određuje donji položaj 3D elemeta
  • backface-visibility - definiše da li će element biti vidljiv

3D transform metode:

TRANZICIJE

Tranzicija se koristi da ublaži naglu promenu vrednosti izabranog CSS svojstva u određenom vremenskom periodu, tako što postepeno menja vrednosti CSS svojstva.

Kod tranzicije razlikujemo 4 osobine:

  • transition-property

    -definiše na kome CSS svojstvu će biti izvršena tranzicija.

  • transition-duration

    -definiše trajanje tranzicije

  • transition-timing-function

    -definiše funkciju koja će biti zadužena za brzinu tranzicije u vremenu.

    • linear – predefinisana tranzicija sa konstntnom brzinom
    • ease – predefinisana tranzicija koja ubrzava kroz sredinu pa usporava na kraju
    • ease-in – predefinisana tranzicija koja počinje sporo pa zatim ubrzava sve do kraja
    • ease-out - predefinisana tranzicija koja počinje velikom brzinom a zatim usporava do kraja
    • ease-in-out - predefinisana tranzicija koja počinje sporo, nakon čega ubrzava u sredini a zatim usporava do kraja
    • cubic-bezier(x1,y1,x2,y2) - programer sam definiše ponašanje funkcije
  • transition-delay

    -definiše vreme nakon kojeg će početi tranzicija od trenutka aktiviranja tranzicije

Skracena sintaxa za transition

.selector { transition: [property] [duration] [timing-function] [delay];}


Pokretanje tranzicije moze biti izvrseno na sledece nacine:

  • pseudo klasama
    • :hover – kada mišem predjemo preko elementa
    • :focus – kada je element u fokusu
    • :active – kada je link element aktivan tj. kliknut
  • menjanjem svojstva ili dodavanje klase koja sadrži promenu svojstva sa JavaScriptom.

ANIMACIJE

Animacija se zasniva na korišćenju sledećih svojstava:

  • animation-name

    -Ovo svojstvo definiše ime @keyframes pravila po kome će se izvoditi animacija. CSS svojstvo @keyframes kontroliše medju-korake u animacijskoj sekvenci definisanjem stilova u toku trajanja animacije. Koristeći @keyframes možemo da definišemo tajming za primenu odredjenih CSS svojstava tokom sekvence animacije, 0% označava prvi trenutak animacije, dok 100% pokazuje krajnje stanje animacije.

  • animation-duration

    Trajanje jednog ciklusa animimacije. Inicjalna vrednost svojstva je 0s

  • animation-timing-function

    Definisanje funkcije koja odredjuje brzinu izvršavanja u toku vremena. Inicjalna vrednost svojstva je ease.

    • linear - predefinisana tranzicija sa konstntnom brzinom

    • ease - predefinisana tranzicija koja ubrzava kroz sredinu pa usporava na kraju

    • ease-in - predefinisana tranzicija koja počinje sporo pa zatim ubrzava sve do kraja.

    • ease-out - predefinisana tranzicija koja počinje velikom brzinom a zatim usporava do kraja.

    • ease-in-out - predefinisana tranzicija koja počinje sporo, nakon čega ubrzava u sredini a zatim usporava do kraja.

    • cubic-bezier(x1,y1,x2,y2) - programer sam definiše ponašanje funkcije.

  • animation-delay

    Definiše vreme nakon kojeg će početi tranzicija od trenutka aktiviranja tranzicije. Inicjalna vrednost svojstva je 0s

  • animation-direction

    Definiše smer izvršavanja animacije u odnosu na zadate podatke. Nakon svokog ciklusa se resetuje i ponavlja. Inicjalna vrednost svojstva je normal.

    • normal - od definisanog početka pa do kraja.

    • reverse - od definisanog kraja ka početku.

    • alternate - prvo normalno do kraja pa onda reverse do početka.

    • alternate-reverse - prvo unazad do početka pa zatim unapred do kraja.

  • animation-iteration-count

    Broj ciklusa animacija. Inicjalna vrednost svojstva je 1. Pored odredjenog broja može biti i “infinite”.

  • animation-fill-mode

    definise koje vrednosti se primenjuju pre / posle animacije.Inicjalna vrednost svojstva je none.

      none - lement je podešen na njegovo podrazumevano mesto pre pocetka animacije i vracca se u to podrazumevano mesto nakon sto se animacija zavrsi.

      forwards - element ostaje na mestu definasanom za kraj animacije.

      backwards - mesto elementa će se već primenjivati pre nego što se animacija zapravo pokrene.

      both - nimacija će prihvati i forwards i backwards.
